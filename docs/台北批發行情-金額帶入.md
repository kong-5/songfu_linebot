# 金額帶入台北果菜市場批發行情

## 功能說明

在**訂單整理**新增紙本訂單時，辨識出品項後可一鍵**帶入台北批發價**，將「金額」欄位填入**台北果菜市場（台北一、台北二）**的批發行情，作為參考單價。

- **資料來源**：農業部農業資料開放平臺 — [農產品交易行情](https://data.moa.gov.tw/open_detail.aspx?id=037)（AMIS）
- **價格意義**：**平均價（元/公斤）**，每日由批發市場回報
- **更新頻率**：每日約 6、8、10、11、12、22 時更新

## 使用方式

1. 進入 **物流工具 → 訂單整理 → 新增一筆訂單**
2. 輸入或辨識出訂單內容後，在預覽品項區塊點選 **「帶入台北批發價」**
3. 系統會依**訂單日期**向農業部 API 取得當日台北一、台北二行情，並依**品名**自動對應（完全相符或品名與作物名稱相互包含）
4. 有對應到的品項，**金額**欄位會填入該品項的批發平均價（元/公斤），可再手動修改後儲存

## 注意事項

- 若查詢日期為**休市**或資料尚未更新，可能無法取得行情
- 品名與農業部「作物名稱」不完全一致時，可能對應不到；可手動填寫金額或稍後再試
- 金額欄位儲存的是**參考單價（元/公斤）**，僅供採購／成本估算參考，不取代實際交易價格
