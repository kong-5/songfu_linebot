# LINE 機器人：啟動與排程

## 後台位置

左側選單最上方 **「LINE 機器人」→「啟動與排程」**（`/admin/line-bot`）。

## 運作模式

| 模式 | 說明 |
|------|------|
| **一律開啟** | 測試或全天候收單，不受時間限制（預設建議） |
| **一律關閉** | 不回覆叫貨相關訊息（圖片／文字皆略過） |
| **依下方時段** | 以**台北時間**判斷是否在「開始～結束」之間（可跨日，例如 18:00～03:00） |

## AI 過濾（選用）

勾選後，對**非**「收單關鍵字／收單中／結單／今日訂單」等已內建處理的訊息，會先以 **Google Gemini** 判斷是否與叫貨有關；若判定為閒聊則**不回覆**。需設定 `GOOGLE_GEMINI_API_KEY`。

測試階段建議**先關閉**，避免誤擋。

## 紀錄列表

儲存設定時會寫入「設定與狀態紀錄」表格，方便追蹤變更。

## Gemini 與 Claude

本專案已整合 **Google Gemini**，與既有後台 AI、可選圖片流程一致。**Claude** 需另接 Anthropic API 與額外開發；若未來要支援可再規劃。
